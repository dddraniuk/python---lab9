# python---lab9
Формати PNG, JPG та SVG відрізняються способом збереження та якістю зображення. PNG — растровий формат без втрат, добре підходить для графіків і зберігає чіткість ліній, але має більший розмір файлу. JPG використовує стиснення з втратами, тому займає менше місця, але створює артефакти та розмиває тонкі елементи, що робить його менш придатним для наукових побудов. SVG — векторний формат, який масштабується без втрати якості та найкраще передає графіки, схеми й діаграми. Отже, для точних наукових зображень оптимальним є SVG, для веб-використання — PNG, а JPG підходить лише коли потрібен мінімальний розмір файлу.
